Projects

KamayBayin

An Android-Based Real-Time Filipino Sign Language to Baybayin Translation Application Through Digital Image Processing Using Convolutional Neural Network

KamayBayin is an Android-based learning application for Filipino Sign Language (FSL) and Baybayin. It supports recognition of the FSL alphabet and 75 basic static and dynamic words or phrases using real-time camera-based assessment. The system uses a MobileNetV3 model deployed through LiteRT (TensorFlow Lite) to enable efficient on-device gesture recognition. FSL and Baybayin lessons are delivered through interactive tutorials and quizzes, with user progress stored and visualized using Firebase. The application is designed for educational use and supports offline sign recognition.
